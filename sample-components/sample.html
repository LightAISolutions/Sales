Sample auto-refresh page

This page polls its own version file every 10 seconds. When the version in html-versions/samplehtml.version.txt changes, the page reloads automatically. The pill in the bottom-right shows the current version and the polling state.

Bump the version (e.g. |v01.00w| → |v01.01w|), redeploy, and within ~10 seconds the open page will reload to pick up the new build.
